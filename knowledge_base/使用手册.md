## 使用场景

### 场景1：审核新产品包装

**步骤**：
1. 确定产品类别（饮料、乳制品等）
2. 查阅 `2_industry_standards/` 中的对应标准
3. 检查执行标准号是否正确
4. 对照 `1_national_standards/` 中的国家标准
5. 使用 `3_advertising_law/禁用词清单.md` 检查文案
6. 生成审核报告

### 场景2：检查营养成分表

**步骤**：
1. 打开 `1_national_standards/GB28050_营养标签通则.pdf`
2. 检查单位是否符合规范（g、mg、kJ等）
3. 验证能量换算公式
4. 检查修约间隔

### 场景3：检查广告文案

**步骤**：
1. 打开 `3_advertising_law/禁用词清单.md`
2. 逐项检查包装上的宣传语
3. 参考 `3_advertising_law/合规指南.md` 中的允许表述
4. 提出修改建议

### 场景4：验证企业信息

**步骤**：
1. 查阅 `4_brand_assets/[企业名称]/企业信息.md`
2. 对比包装上的企业名称、地址、电话
3. 检查是否与品牌资产库一致

## 常见问题

### Q1: 如何判断产品的执行标准？

**A**: 
1. 根据产品名称判断产品类别
2. 查阅 `2_industry_standards/搜集指南.md` 中的产品类别表
3. 找到对应的执行标准号
4. 验证包装上的标准号是否匹配

### Q2: 营养成分表的单位有什么要求？

**A**: 
- 糖、脂肪、蛋白质、碳水：必须用 **g**
- 钠：必须用 **mg**
- 能量：必须用 **kJ**
- 详见 `1_national_standards/GB28050_营养标签通则.pdf`

### Q3: 包装上可以使用"最好"吗？

**A**: 
- ❌ 不可以，"最"是禁用词
- ✅ 可以改为"优质"、"精选"等相对性表述
- 详见 `3_advertising_law/禁用词清单.md`

### Q4: 如何上传品牌资产？

**A**: 
1. 复制 `4_brand_assets/品牌资产模板.md`
2. 按照模板填写企业信息
3. 上传到 `4_brand_assets/[企业名称]/` 文件夹
4. 将图片资产存储在 `images/` 子文件夹

### Q5: 知识库多久更新一次？

**A**: 
- 季度更新：每季度末检查一次
- 紧急更新：如有重大法规变动，立即更新
- 下次更新日期：2026年Q3
