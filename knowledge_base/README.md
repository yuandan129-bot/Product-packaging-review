# 包装审核知识库

本知识库为 AI 包装审核助手的核心数据支撑，按季度更新。

## 文件夹结构

```
knowledge_base/
├── 1_national_standards/          # 国家标准库
│   ├── GB7718_食品标签通则.pdf
│   ├── GB28050_营养标签通则.pdf
│   └── GB2760_食品添加剂使用标准.pdf
│
├── 2_industry_standards/          # 行业分类库
│   ├── 饮料/
│   │   ├── GB_T_10792_碳酸饮料.pdf
│   │   └── GB_T_29602_果汁饮料.pdf
│   ├── 乳制品/
│   │   ├── GB_25191_巴氏杀菌乳.pdf
│   │   └── GB_25192_灭菌乳.pdf
│   ├── 糖果/
│   │   └── GB_T_19343_巧克力.pdf
│   └── 烘焙食品/
│       └── GB_T_20981_面包.pdf
│
├── 3_advertising_law/             # 广告法与营销词库
│   ├── 禁用词清单.txt
│   ├── 虚假宣传识别规则.md
│   └── 市场监管总局_最新规定.pdf
│
└── 4_brand_assets/                # 品牌资产库（用户自维护）
    ├── 品牌资产模板.xlsx
    └── 示例企业/
        ├── 企业信息.json
        └── 产品分类.json
```

## 更新周期

- **季度更新**：每季度末检查标准是否有新版本发布
- **紧急更新**：如有重大法规变动，立即更新

## 数据来源

| 知识库 | 官方来源 |
|------|--------|
| 国家标准 | [GB官网](http://www.gb688.cn/) / 标准化委员会 |
| 行业标准 | 相关行业协会 / 标准化委员会 |
| 广告法 | [市场监管总局](http://www.samr.gov.cn/) / 工商总局 |
| 品牌资产 | 用户自行维护 |

## 使用说明

- 所有 PDF 文档应保持原始格式，便于版本追溯
- 文本文件（.txt/.md）用于快速查询和规则定义
- JSON 格式用于结构化数据（品牌资产）
