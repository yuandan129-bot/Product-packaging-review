# 中国广告法禁用词清单

## 概述

根据《中华人民共和国广告法》（2015年修订版）及市场监管总局相关规定，食品包装上禁止使用以下词汇和表述。

---

## 一、绝对化用语禁用词

### 1.1 最高级词汇

| 禁用词 | 示例 | 说明 |
|------|------|------|
| 最 | 最好、最优、最先进 | 任何"最"字表述 |
| 第一 | 第一品牌、第一选择 | 包括"首个"、"首家" |
| 唯一 | 唯一选择、唯一推荐 | 绝对唯一性表述 |
| 独一无二 | 独一无二的配方 | 同上 |
| 冠军 | 销售冠军、品质冠军 | 竞争性排名 |
| 领先 | 行业领先、技术领先 | 相对比较性表述 |
| 顶级 | 顶级品质、顶级工艺 | 最高等级表述 |
| 极致 | 极致口感、极致体验 | 极端化表述 |

### 1.2 绝对性表述

| 禁用词 | 示例 | 说明 |
|------|------|------|
| 永远 | 永远新鲜、永远美味 | 绝对时间表述 |
| 完全 | 完全天然、完全无添加 | 绝对完整性表述 |
| 绝对 | 绝对安全、绝对健康 | 绝对保证表述 |
| 100% | 100%纯天然、100%无污染 | 绝对百分比 |
| 全部 | 全部采用进口原料 | 绝对全面性 |

---

## 二、虚假宣传禁用词

### 2.1 功效夸大

| 禁用词 | 示例 | 说明 |
|------|------|------|
| 治疗 | 治疗便秘、治疗高血压 | 医疗功效表述 |
| 治愈 | 治愈糖尿病 | 医疗治愈表述 |
| 特效 | 特效降脂、特效减肥 | 特殊功效表述 |
| 神奇 | 神奇功效、神奇配方 | 夸大功效 |
| 奇迹 | 创造奇迹、奇迹般效果 | 不可思议的效果 |
| 秘方 | 祖传秘方、独家秘方 | 神秘性夸大 |
| 专利 | 专利配方、专利技术 | 未获得专利的表述 |

### 2.2 来源虚假

| 禁用词 | 示例 | 说明 |
|------|------|------|
| 进口 | 进口原料（未实际进口） | 虚假产地 |
| 原产地 | 原产地法国（实际非法国） | 虚假原产地 |
| 有机 | 有机产品（未认证） | 虚假认证 |
| 绿色 | 绿色食品（未认证） | 虚假认证 |
| 无污染 | 完全无污染 | 虚假安全承诺 |

## 三、医疗保健禁用词

### 3.1 医疗相关

| 禁用词 | 示例 | 说明 |
|------|------|------|
| 医学证明 | 医学证明有效 | 医学权威表述 |
| 临床验证 | 临床验证有效 | 医学验证表述 |
| 药物 | 本产品如同药物 | 药物性表述 |
| 处方 | 医生处方推荐 | 医疗处方表述 |
| 疾病 | 预防疾病、治疗疾病 | 疾病相关表述 |

### 3.2 保健功能

| 禁用词 | 示例 | 说明 |
|------|------|------|
| 保健 | 保健功能（未备案） | 未获批的保健功能 |
| 增强免疫力 | 增强免疫力（未备案） | 保健功能表述 |
| 改善睡眠 | 改善睡眠（未备案） | 保健功能表述 |
| 延年益寿 | 延年益寿 | 夸大保健功能 |

## 九、检查清单

在审核包装文案时，逐项检查：

- [ ] 是否包含"最"、"第一"、"唯一"等最高级词汇
- [ ] 是否包含"永远"、"完全"、"绝对"等绝对性表述
- [ ] 是否包含医疗、治疗、治愈等医疗相关词汇
- [ ] 是否包含虚假的功效声称
- [ ] 是否包含虚假的来源或原产地信息
- [ ] 是否包含虚假的权威性表述
- [ ] 是否包含虚假的统计数据或销售数据
- [ ] 是否包含不实的比较表述
